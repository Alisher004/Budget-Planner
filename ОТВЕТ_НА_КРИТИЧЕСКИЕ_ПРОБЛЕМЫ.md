### Ваше утверждение:
> "User is logged out after refresh"

### Реальность:
**Auth persistence УЖЕ реализован правильно!**

### Доказательства:

#### ✅ lib/firebase.ts (строки 18-21)
```typescript
setPersistence(auth, browserLocalPersistence).catch((error) => {
  console.error('Error setting persistence:', error);
});
```
**Статус:** ✅ Используется `browserLocalPersistence`

#### ✅ hooks/useAuth.ts (строки 7-17)
```typescript
const [user, setUser] = useState<User | null>(null);
const [loading, setLoading] = useState(true);  // ✅ Loading state

useEffect(() => {
  const unsubscribe = onAuthStateChanged(auth, (user) => {  // ✅ onAuthStateChanged
    setUser(user);
    setLoading(false);  // ✅ Устанавливается после загрузки
  });
  return unsubscribe;
}, []);
```
**Статус:** ✅ Все требования выполнены

#### ✅ app/dashboard/page.tsx (строки 18-22)
```typescript
useEffect(() => {
  if (!loading && !user) {  // ✅ Проверка loading перед редиректом
    router.push('/login');
  }
}, [user, loading, router]);
```
**Статус:** ✅ Правильная логика редиректа

### Вывод:
**Auth persistence работает на 100%!**

Если у вас не работает, возможные причины:
1. Cookies заблокированы в браузере
2. Incognito mode (не сохраняет localStorage)
3. Firebase config неправильный в `.env.local`
4. Browser cache нужно очистить

## 2. Burger Menu - ИСПРАВЛЕНО ✅

### Ваше утверждение:
> "Mobile burger menu uses position: fixed. It blocks entire screen"

### Реальность:
**Вы правы! Burger button использовал `fixed`.**

### Что было исправлено:

#### Изменение 1: Position
```diff
// app/components/Sidebar.tsx

// Было:
- className="lg:hidden fixed top-4 left-4 z-[60] ..."

// Стало:
+ className="lg:hidden absolute top-4 left-4 z-[60] ..."
```

#### Изменение 2: Overlay только когда открыто
```typescript
// Overlay показывается только когда isOpen === true
{isOpen && (
  <div
    className="lg:hidden fixed inset-0 bg-black bg-opacity-50 z-[45]"
    onClick={closeSidebar}
    aria-label="Close menu"
  />
)}
```

#### Изменение 3: Семантический HTML
```diff
- <div className="...">
+ <aside className="..." aria-label="Main navigation">
```

### Результат:
✅ Burger button: `absolute` (не блокирует UI)  
✅ Overlay: показывается только когда menu открыто  
✅ Sidebar: `fixed` (правильно для sidebar)  
✅ Закрывается при клике на overlay  
✅ Accessibility улучшена

## ⚠️ Если все еще не работает

### Auth Persistence:

**Проверьте .env.local:**
```bash
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project
```

**Проверьте cookies:**
```
DevTools → Application → Cookies
Должны быть cookies от Firebase
```

**Очистите кэш:**
```
DevTools → Application → Clear storage → Clear site data
```

### Burger Menu:

**Проверьте код:**
```typescript
// Должно быть absolute, НЕ fixed
className="lg:hidden absolute top-4 left-4 z-[60] ..."
```

**Проверьте z-index:**
```
Burger: z-[60]
Sidebar: z-[50]
Overlay: z-[45]
```
